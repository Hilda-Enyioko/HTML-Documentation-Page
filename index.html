Introduction

HyperText Markup Language(HTML) is the standard markup language for designing documents to be displayed in a web browser. It describes the structure of a Web page. It allows the creation and structure of sections, paragraphs, and links using HTML elements (the building blocks of a web page) as well as tags and attributes. HTML files have a .html extension (e.g. index.html).

HTML Elements

HTML elements are the building blocks of a web page. They are tools that are used to create webpages and design the content of the webpage. They are identified by a start tag, some content, and an end tag. A start tag is represented by two angle brackets with the symbol of the element in their middle i.e. <h1> for the Heading1 element. An end tag is represented by an angle bracket, followed by a slash, then a closing angle bracket i.e. </h1>for the Heading1 element In between the start and the end tag is the content of the element i.e. <h1>Heading1 Content</h1> Note: It is not all elements require an end tag.
