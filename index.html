HTML Documentation Page

Introduction

HyperText Markup Language(HTML) is the standard markup language for designing documents to be displayed in a web browser. It describes the structure of a Web page. It allows the creation and structure of sections, paragraphs, and links using HTML elements (the building blocks of a web page) as well as tags and attributes. HTML files have a .html extension (e.g. index.html).

HTML Elements

HTML elements are the building blocks of a web page. They are tools that are used to create webpages and design the content of the webpage. They are identified by a start tag, some content, and an end tag. A start tag is represented by two angle brackets with the symbol of the element in their middle i.e. <h1> for the Heading1 element. An end tag is represented by an angle bracket, followed by a slash, then a closing angle bracket i.e. </h1>for the Heading1 element In between the start and the end tag is the content of the element i.e. <h1>Heading1 Content</h1> Note: It is not all elements require an end tag.

Examples of HTML Elements
Element Representation Function
Heading 1 <h1></h1> This is the highest level of heading in HTML, typically used for the main heading of a page. Heading elements should be used in a logical hierarchy.
Paragraph <p></p> This is used to define a paragraph of text.
Image <i> This is used to add an image to the content of a webpage.
Anchor <a></a> This is used to create hyperlinks or clickable links in HTML.
Line Break <br> This allows you to create a line break within a block of text, forcing subsequent content to appear on a new line.

HTML Attributes

HTML attributes are additional properties pecified within the start tag of an HTML element that provides extra information or modify their behavior in aspects such as appearance, behavior, data, or relationships of the element. The format of an HTML attribute consists of two parts: the attribute name and its value. Both of which are specified within the opening tag of an Element. The attribute name is the written in lower case, followed by an equal to sign, and then the attribute's value i.e. <h1 class="title"> for the class attribute for the Heading1 element.

Examples of HTML Attributes
Attribute Representation Function
Class <div class="class-value"></div> This specifies one or more CSS classes to apply to an element for targeting of specific elements for styling.
ID <h1 id="id-value"></h1> This provides a unique identifier for an element.
Source <i src="image-url"> This specifies the source URL of an external resource.
Href <a href="hyperlink"></a> This defines the URL or destination of a link when the element represents a hyperlink.
Style <p style="CSS-style"></p> This allows inline CSS styling to be applied directly to an element.
