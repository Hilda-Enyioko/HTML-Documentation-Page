Introduction

HyperText Markup Language(HTML) is the standard markup language for designing documents to be displayed in a web browser. It describes the structure of a Web page. It allows the creation and structure of sections, paragraphs, and links using HTML elements (the building blocks of a web page) as well as tags and attributes. HTML files have a .html extension (e.g. index.html).
